Team WT House

Product Owner : Jing Ting Huang
Scrum Master, Developer : Wei Hao Wu
Main Developer, GitHub Manager : Chun Jou Lin

Team WT House is responsible for building up the Web App to collect the annotated text and images from users. The annotation page collects the annotated data by letting users input what they see in the images and based on the comparision function to extract the legible image. The validation page collects the validated data by letting users evaluate the modeling results and based on evaluation function to validate the model results.
